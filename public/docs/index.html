Evaluation Package Overview

This page explains what development teams receive when testing SentariX Prime and how the evaluation package is structured.

What the package includes

Approved teams receive a complete set of build files and supporting materials needed to explore SentariX Prime in a real environment.

Platform libraries

Prebuilt library files for supported platforms Windows includes a DLL Linux includes a SO macOS support is available upon request

Python wheels

Ready to use Python wheels for common versions Each wheel contains the native extension required for evaluation

C Sharp binding

A lightweight C Sharp file that connects to the native library and exposes the public interface

Example projects

Small examples for Python and C Sharp that show how to call the public interface during evaluation

Integration guide

A private PDF that explains how to connect Prime to a real session and how to read the public output

Supported languages

SentariX Prime works in any environment that can call a native library. Official bindings are provided for the following languages.

Python

Distributed as wheels that include the required extension Works on Windows and Linux with common versions

C Sharp

Provided through a thin wrapper that calls the native library Can be used in engines and custom applications

Other languages

Any environment that supports native calls can integrate Prime Additional bindings can be arranged during evaluation

What teams can expect

The evaluation build is designed to drop into existing systems with minimal effort. Teams typically test Prime inside movement logic, input feel, interpolation, simulation steps, or other timing sensitive areas.

Fast setup

Each package includes clear examples that can be adapted in minutes

Works in real environments

The build is made for hands on evaluation inside actual products or prototypes

Simple interface

Prime returns one public value that teams use wherever consistent feel is required

Next steps

Access to the evaluation package is provided upon request. Teams receive the full set of files and the private integration guide once approved.
